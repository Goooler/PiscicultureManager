# PiscicultureManager
## 渔业养殖管理系统


### 水产养殖的几个参数如下：
| 参数       | 溶氧量  | PH值  | 氨氮       | 亚硝酸盐 | 余氯      | 温度   |
| ---------- | ------- | ----- | ---------- | -------- | --------- | ------ |
| 合适的区间 | 5~8mg/L | 6.5~8 | 0~0.015ppm | 0~0.5ppm | 0~0.02ppm | 20~40℃ |

### 拟定要展示并调整的几个参数：温度、溶氧量、PH、亚硝酸盐、氨氮
